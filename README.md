# InstantBlog

InstantBlog strives to be a simple platform for blogging - based on a Java Spring Backend and a TypeScript React Frontend.

Bloggers are able to manage their blog and post new blogposts.

## Features

- Bloggers can login and use the Editor page to write new posts.
- Images can be added if you provide a valid IMG url.
- Maximal character count with live-updated progress bar.
- H2 DB in the backend to save posts & more.
- Communication between Frontend and Backend via REST API.
- Spring Security with JWT Token (saved in Local Storage in Frontend).
- Submitting new posts only allowed with correct JWT token and role "ROLE_BLOGGER".
